hexo博客发布工具使用简介

通过github在hexo上搭建个人博客的使用说明连接：https://www.jianshu.com/p/390f202c5b0e

1、在source的__post目录下添加需要发布的md文章（md文件撰写可通过typero）

2、完成文件撰写后，在对应的目录文件夹下先执行

hexo clean 删除以前的残留

hexo g 生成md文件对应的网页静态文件

hexo s &后台运行hexo server服务器，提供本地网页内容查看，查看网址为http://localhost:4000

hexo d 将就绪的博客发布到github上对应的个人主义上
